Chess Puzzle Trainer

A lightweight, mobile-responsive web app for randomized chess puzzle training.

This tool pulls curated puzzles from Lichess studies based on difficulty levels (Easy, Medium, Hard, and Mate in 2).

Features:
Direct Lichess Integration: Launches interactive puzzles directly in the Lichess study interface.
Randomized Selection: Uses cryptographic-grade randomness to choose puzzles.
Optimized for Tablet/Mobile: Custom-themed UI designed for quick training sessions on the go.
